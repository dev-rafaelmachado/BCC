Fole Follow

Jokenpô Python

Trabalho - R. Algoritmo
Simulador de Jokenpô onde o jogador poderá escolher entre jogar contra a maquina, contra outra pessoa ou apenas assistir maquina vs maquina jogarem. O código é todo em Python utilizando elementos básicos como If e While.


Máquina de Bebidas

Trabalho - R. Algoritmo
O projeto simula uma máquina de bebidas onde o usuário poderá escolher dentre vários produtos para comprar, para pagar o usuário tera de inserir notas e moedas e o programa devolverá em notas e moedas o seu troco. O código é todo em Python utilizando elementos If, While, Array, Matriz.

Smart Sinal

TDE Smart City - Sist. Cyberfísicos
Sinal Inteligente que por meio de um sensor de ultra-som reconhece se um pedestre deseja atravessar a rua, além de um buzer para sinalização para pessoas cegas e/ou baixa visão trazendo mais segurança e praticidade ao trânsito. O projeto utiliza um Arduino 1 sensor de ultra-som, buzzer e leds para simular o sinal.
